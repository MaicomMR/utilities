
## 1. Install Oh My Zsh
`sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"`

## 2. Install theme fonts
`git clone https://github.com/powerline/fonts.git`

`cd fonts`

`./install.sh`

## 3. Change Theme
`$ open ~/.zshrc`
Set `ZSH_THEME="agnoster"` and save the file
